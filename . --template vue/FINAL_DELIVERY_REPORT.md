## 🚀 部署说明

### 系统要求
- Windows 7 及以上
- 内存：500 MB 以上
- 磁盘：100 MB 以上

### 安装方式
1. 运行 `run.bat` 或直接执行 `npx electron dist-electron/main.js`
2. 首次运行会自动加载 Excel 数据
3. 应用数据存储在 `data.json` 中

### 运行环境
```bash
# 安装依赖
npm install

# 开发模式
npm run dev

# 生产构建
npm run build-nosign

# 运行应用
npx electron dist-electron/main.js
```
